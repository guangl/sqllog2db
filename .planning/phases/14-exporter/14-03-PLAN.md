    实现逻辑（按 14-RESEARCH.md Pattern 3 + 14-PATTERNS.md csv 一节）：

    1. **D-09 路径推导**：`let base_path: &Path = final_path.unwrap_or(self.path.as_path());` 然后 `let stem = base_path.file_stem().unwrap_or_default();` 然后 `let companion = base_path.with_file_name(format!("{}_templates.csv", stem.to_string_lossy()));`。`companion` 类型为 `PathBuf`。

    2. **目录确保**：`ensure_parent_dir(&companion).map_err(|e| Error::Export(ExportError::WriteFailed { path: companion.clone(), reason: format!("create dir failed: {e}") }))?;`

    3. **D-10 始终覆盖**：用 `std::fs::File::create(&companion)`（不用 OpenOptions append），错误用 `Error::Export(ExportError::WriteFailed { path: companion.clone(), reason: format!("create companion failed: {e}") })` 包装。

    4. **BufWriter**：`let mut writer = std::io::BufWriter::new(file);`（伴随文件数据量小，无需 16MB 大缓冲）。

    5. **表头**：`writer.write_all(b"template_key,count,avg_us,min_us,max_us,p50_us,p95_us,p99_us,first_seen,last_seen\n").map_err(|e| Error::Export(ExportError::WriteFailed { path: companion.clone(), reason: format!("write header failed: {e}") }))?;`

    6. **数据行写入**（每行预分配 `Vec<u8>` line_buf，循环 clear 复用；模板示例见 14-PATTERNS.md csv.rs 段）：
       - 准备 `let mut itoa_buf = itoa::Buffer::new();` 和 `let mut line_buf: Vec<u8> = Vec::with_capacity(512);`
       - 遍历 `for s in stats`：
         - `line_buf.clear();`
         - template_key：用 CSV 转义 + 双引号包裹（参照 14-PATTERNS.md "CSV 转义" 段）：`line_buf.push(b'"'); write_csv_escaped(&mut line_buf, s.template_key.as_bytes()); line_buf.push(b'"'); line_buf.push(b',');`
         - count / avg_us / min_us / max_us / p50_us / p95_us / p99_us（u64）：`line_buf.extend_from_slice(itoa_buf.format(s.count).as_bytes()); line_buf.push(b',');` 重复 7 次（每个数值字段一次）
         - first_seen / last_seen（普通时间字符串，逗号分隔；按 RESEARCH 建议直接写：`line_buf.extend_from_slice(s.first_seen.as_bytes()); line_buf.push(b','); line_buf.extend_from_slice(s.last_seen.as_bytes());`）
         - 行尾：`line_buf.push(b'\n');`
       - 写入：`writer.write_all(&line_buf).map_err(|e| Error::Export(ExportError::WriteFailed { path: companion.clone(), reason: format!("write row failed: {e}") }))?;`

    7. **显式 flush**：`writer.flush().map_err(|e| Error::Export(ExportError::WriteFailed { path: companion.clone(), reason: format!("flush failed: {e}") }))?;`

    8. **日志**：`info!("Template companion CSV written: {}", companion.display());`

    9. 返回 `Ok(())`。
