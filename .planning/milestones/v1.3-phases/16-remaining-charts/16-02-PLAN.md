# Plan 16-02: ChartsConfig 配置扩展

## Goal

在 `ChartsConfig` 新增 `trend_line` 和 `user_pie` 两个 bool 字段（默认 true），
更新 `Default` impl，并更新 `init` 命令生成的 TOML 模板。

## Dependency

依赖 Plan 16-01 完成（compile 通过后才执行本计划）。

## Tasks

### Task 1: 扩展 ChartsConfig

文件: `src/features/mod.rs`

在 `ChartsConfig` struct 末尾追加两个字段：

```rust
/// 是否生成时间趋势折线图（默认 true）
#[serde(default = "default_true")]
pub trend_line: bool,
/// 是否生成用户占比饼图（默认 true）
#[serde(default = "default_true")]
pub user_pie: bool,
```

更新 `Default` impl：

```rust
impl Default for ChartsConfig {
    fn default() -> Self {
        Self {
            output_dir: "charts/".to_string(),
            top_n: 10,
            frequency_bar: true,
            latency_hist: true,
            trend_line: true,
            user_pie: true,
        }
    }
}
```

### Task 2: 更新 tests

在 `test_charts_config_default_values` 中追加断言：
```rust
assert!(cfg.trend_line);
assert!(cfg.user_pie);
```

新增反序列化测试：
```rust
#[test]
fn test_charts_config_deserialize_trend_user_flags() {
    let toml = r#"
output_dir = "out/"
trend_line = false
user_pie = false
"#;
    let cfg: ChartsConfig = toml::from_str(toml).unwrap();
    assert!(!cfg.trend_line);
    assert!(!cfg.user_pie);
    assert!(cfg.frequency_bar); // 默认 true
    assert!(cfg.latency_hist);  // 默认 true
}

#[test]
fn test_charts_config_deserialize_only_output_dir_has_new_defaults() {
    let cfg: ChartsConfig = toml::from_str(r#"output_dir = "out/""#).unwrap();
    assert!(cfg.trend_line);
    assert!(cfg.user_pie);
}
```

### Task 3: 更新 init 命令 TOML 模板

查找 `src/cli/init.rs`（或 init 相关文件）中生成 `[features.charts]` 段的字符串，
追加新字段说明：

```toml
[features.charts]
output_dir = "charts/"
top_n = 10
frequency_bar = true
latency_hist = true
trend_line = true    # 时间趋势折线图
user_pie = true      # 用户/Schema 执行占比饼图
```

如果 init.rs 不存在此段（图表是可选配置），则跳过此步骤，仅更新 struct。
