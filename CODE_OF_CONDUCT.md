# 贡献者行为守则

本项目采纳 Contributor Covenant 行为守则。所有参与者应遵守此守则。

## 我们的承诺

为了促进开放和欢迎的环境，我们作为贡献者和维护者承诺：

- 在参与本项目及社区时，确保所有人获得无骚扰的体验，无论其年龄、体型、残疾、种族、性别认同、表达、经验水平、国籍、个人外貌、种族、宗教或性认同和性取向。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用欢迎和包容的语言
- 对不同观点和经历抱有尊重态度
- 虚心接受建设性批评
- 关注社区最佳利益
- 向其他社区成员表现同情

不可接受的行为包括：

- 使用性别歧视、性暗示或冒犯性的语言或意象
- 人身攻击、侮辱或贬低评论
- 公开或私下骚扰
- 发布他人的私人信息（如地址、电话、电子邮件）而未明确许可
- 其他可合理地认为在专业环境中不适当的行为

## 执法

项目维护者负责澄清可接受行为的标准，并对任何不可接受的行为采取适当的纠正行动。

项目维护者有权删除、编辑或拒绝不符合本行为守则的评论、提交、代码、wiki 编辑、问题和其他贡献。

对被禁止行为的报告可通过 [guangl@example.com](mailto:guangl@example.com) 联系项目团队。所有投诉将被审查和调查，并做出适当的回应。
