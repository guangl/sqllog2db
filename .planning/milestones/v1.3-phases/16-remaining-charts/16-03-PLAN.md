# Plan 16-03: 时间趋势折线图

## Goal

新建 `src/charts/trend_line.rs`，实现 `draw_trend_line()` 函数，
生成 `frequency_trend.svg`：X 轴小时粒度时间戳，Y 轴该小时内 SQL 执行总次数，折线连续。

## Dependency

依赖 Plan 16-01（`iter_hour_counts()` 可用）、Plan 16-02（`ChartsConfig.trend_line` 可用）。
本文件是全新创建，不修改现有文件，可与 16-04 并行执行。

## Design Decisions

- 图表尺寸：1200×600（与现有图表一致）
- X 轴：按 hour bucket key 升序，标签格式：
  - 若 span 为单天：显示 `"HH:00"` (e.g. `"10:00"`)
  - 若 span 跨多天：显示 `"MM-DD HH:00"` (e.g. `"01-15 10:00"`)
  - 通过比较 buckets 首尾 bucket 的日期前缀（前 10 字符）判断
- Y 轴：线性，从 0 到 max_count * 1.1（留出上边距）
- 折线颜色：`RGBColor(220, 50, 47)`（红色，区分条形图的蓝色）
- 数据点：每个小时一个点，用 Circle marker 标注

### Task 2: X 轴标签生成逻辑

```rust
fn build_x_labels(hour_counts: &[(&str, u64)]) -> Vec<String> {
    let multi_day = is_multi_day(hour_counts);
    hour_counts
        .iter()
        .map(|(bucket, _)| format_bucket_label(bucket, multi_day))
        .collect()
}

fn is_multi_day(hour_counts: &[(&str, u64)]) -> bool {
    if hour_counts.len() < 2 {
        return false;
    }
    let first_day = &hour_counts[0].0[..10.min(hour_counts[0].0.len())];
    let last_day = &hour_counts[hour_counts.len() - 1].0[..10.min(hour_counts.last().unwrap().0.len())];
    first_day != last_day
}

fn format_bucket_label(bucket: &str, multi_day: bool) -> String {
    // bucket 格式: "2025-01-15 10"
    if multi_day && bucket.len() >= 13 {
        // "MM-DD HH:00"
        format!("{} {:}:00", &bucket[5..10], &bucket[11..13])
    } else if bucket.len() >= 13 {
        // "HH:00"
        format!("{}:00", &bucket[11..13])
    } else {
        bucket.to_string()
    }
}
```

### Task 3: plotters 渲染

使用 `LineSeries` 绘制折线，使用 `Circle` 标注每个数据点：

```rust
chart.draw_series(LineSeries::new(
    counts.iter().enumerate().map(|(i, &c)| (i, c)),
    LINE_COLOR.stroke_width(2),
))?;

chart.draw_series(counts.iter().enumerate().map(|(i, &c)| {
    Circle::new((i, c), 4, LINE_COLOR.filled())
}))?;
```

X 轴使用 `(0..n).into_segmented()` 或 `0usize..n`，
Y 轴使用 `0u64..(max_count * 11 / 10 + 1)` 留出上边距。
