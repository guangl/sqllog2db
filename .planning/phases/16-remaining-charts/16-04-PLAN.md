# Plan 16-04: 用户执行占比饼图

## Goal

新建 `src/charts/user_pie.rs`，实现 `draw_user_pie()` 函数，
生成 `user_schema_pie.svg`：按用户分组显示执行占比饼图，
超过 top_n 的用户归入 "Others" 扇区，长用户名截断处理。

## Dependency

依赖 Plan 16-01（`iter_user_counts()` 可用）、Plan 16-02（`ChartsConfig.user_pie` 可用）。
本文件是全新创建，不修改现有文件，可与 16-03 并行执行。

## Design Decisions

- 图表尺寸：800×600（饼图比条形图更宽高接近）
- 调色板：plotters 内置多色方案，按 HSL 均分色相（避免相邻扇区颜色相近）
- "Others" 扇区颜色：灰色 `RGBColor(150, 150, 150)`
- 用户名截断：20 字符（超出显示 "…"）
- 饼图使用 plotters 的 `draw_series` 绘制扇形（通过角度计算）
- 标签：扇区旁显示 "用户名 (XX.X%)"，扇区过小（<3%）时省略标签

### Task 2: draw_user_pie() 主函数

```rust
pub fn draw_user_pie(
    user_counts: &[(&str, u64)],
    top_n: usize,
    output_path: &std::path::Path,
) -> crate::error::Result<()> {
    if user_counts.is_empty() {
        return Ok(());
    }
    let slices = prepare_slices(user_counts, top_n);
    let total: u64 = slices.iter().map(|s| s.count).sum();
    if total == 0 {
        return Ok(());
    }
    render_pie(&slices, total, output_path)
}
```

### Task 3: 饼图渲染

plotters 原生不提供饼图 series，需要手动计算扇形角度并绘制：

```rust
fn render_pie(
    slices: &[PieSlice],
    total: u64,
    output_path: &std::path::Path,
) -> crate::error::Result<()> {
    let root = SVGBackend::new(output_path, (800, 600)).into_drawing_area();
    root.fill(&WHITE).map_err(|e| to_write_err(output_path, &e))?;

    // 标题
    // 饼图中心和半径
    let cx = 300i32;
    let cy = 280i32;
    let radius = 200i32;

    draw_pie_slices(&root, slices, total, cx, cy, radius, output_path)?;
    draw_legend(&root, slices, total, output_path)?;

    root.present().map_err(|e| to_write_err(output_path, &e))?;
    Ok(())
}
```

扇形绘制使用 `plotters::element::Polygon`，通过角度步进生成扇形顶点列表：

```rust
fn sector_points(cx: i32, cy: i32, r: i32, start_rad: f64, end_rad: f64) -> Vec<(i32, i32)> {
    let mut pts = vec![(cx, cy)];
    let steps = ((end_rad - start_rad).abs() * 30.0) as usize + 2;
    for i in 0..=steps {
        let angle = start_rad + (end_rad - start_rad) * i as f64 / steps as f64;
        let x = cx + (r as f64 * angle.cos()) as i32;
        let y = cy + (r as f64 * angle.sin()) as i32;
        pts.push((x, y));
    }
    pts
}
```
