| Boundary | Description |
|----------|-------------|
| hdrhistogram bucket → plotters 坐标轴 | histogram 内部数据（项目可信），但 value 边界可能导致 log(0) 崩溃 |
| entry.key → SVG 标题文字 | 模板 key 来自 SQL log（外部输入），需字符数限制防止超长标题 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-07 | DoS | log_scale 与非正 min_val | mitigate | `min_val = buckets.first().map(|(v,_)| (*v).max(1)).unwrap_or(1)` 确保下界 ≥ 1；空 histogram 提前返回 |
| T-15-08 | DoS | min_val == max_val 导致 log_scale 范围退化 | mitigate | 检测相等时令 max_val = min_val + 1（或直接提前返回）防止 plotters panic |
